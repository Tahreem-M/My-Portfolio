# Tahreem Malik - Personal Portfolio

A modern, visually stunning personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Dark/Light Mode** - Toggle between themes with preference persistence
- **Responsive Design** - Fully responsive for desktop, tablet, and mobile
- **Smooth Animations** - Scroll-triggered animations and hover effects
- **Sections**:
  - Hero with CTAs (View Projects, Contact Me, Download Resume)
  - About Me with highlights
  - Skills (categorized cards)
  - Projects showcase
  - Experience & Education timeline
  - Contact form with social links

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- Vanilla JavaScript

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser
3. Or use a local server (e.g., `npx serve .` or VS Code Live Server)

## Customization

- **Resume Download**: Update the `initDownloadResume` function in `js/main.js` with your resume PDF path
- **Project Links**: Add your GitHub/live demo URLs to the project cards in `index.html`
- **Theme**: Colors and gradients are defined in CSS custom properties (`:root`) in `css/styles.css`

## Deployment

Deploy to GitHub Pages, Netlify, Vercel, or any static hosting service by uploading the project files.
